Справочник предназначен для отображения курсов валют.

Вкладка "Курсы валют"

Фильтр "Период" позволяет отобразить курсы за определенный день, неделю, месяц, квартал, год или произвольный период.

Фильтр "Валюты" позволяет отобразить курсы либо по выбранным валютам, либо по всем - пункт "(Выделить все)". Также данный фильтр задает валюты, по которым отображаются курсы на текущую дату на домашней странице.

Для применения фильтров необходимо нажать кнопку "Отобразить". По умолчанию на вкладке отображается максимум 45 записей. Переход по страницам осуществляется в навигаторе снизу таблицы.

Для добавления курса необходимо кликнуть либо на иконку в левом верхнем углу таблицы, либо на кнопку "Добавить курс", или нажать клавишу Insert.

Для курсов, добавленных вручную, предусмотрено редактирование и удаление.

Кнопки редактирования и удаления курса расположены в первой колонке таблицы, индивидуально для каждой записи.

При нажатии кнопки "Добавить курс"/"Редактировать курс" откроется диалоговое окно, в котором все поля обязательны для заполнения. Сохранение осуществляется либо по одноименной кнопке, либо клавишей F2.

Вкладка "Средние курсы"

На данной вкладке отображаются средние курсы доллара, евро и фунта стерлинга.

Фильтр "Период" позволяет отобразить средние курсы за определенный день, неделю, месяц, квартал, год или произвольный период.

Фильтр "Валюты" позволяет отобразить средние курсы либо по выбранным валютам, либо по всем - пункт "(Выделить все)".

Для применения фильтров необходимо нажать кнопку "Отобразить".

Вкладка "Кросс-курс"

На данной вкладке отображается динамика кросс-курса валют.

Фильтр "Период" позволяет отобразить кросс-курсы за определенный день, неделю, месяц, квартал, год или произвольный период.

Фильтр "Валютная пара" позволяет задать базовую валюту и валюту котировки. Также данный фильтр задает валюты, по которым отображается кросс-курс на текущую дату на домашней странице.

Чтобы быстро поменять местами валюты, необходимо нажать на кнопку со стрелкой.

Для применения фильтров необходимо нажать кнопку "Отобразить". По умолчанию на вкладке отображается максимум 45 записей. Переход по страницам осуществляется в навигаторе снизу таблицы.

Формула расчета кросс-курса:

Кросс-курс = Курс базовой валюты / Курс валюты котировки

Формула расчета изменения кросс-курса:

Изменение, % = (Кросс-курс за текущий день - Кросс-курс за предыдущий день) * 100% / Кросс-курс за предыдущий день
